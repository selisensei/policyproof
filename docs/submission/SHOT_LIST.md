## Recording notes

- Record at 1440 × 900 or 1440 × 1000 with browser zoom at 100%.
- Close personal tabs and notifications; use only fictional data.
- Keep `.env.local`, terminals containing configuration, and developer tools out of frame.
- Use a production build or deployed URL so no Next.js development indicator is visible.
- Rehearse the deterministic sequence before recording.
- If any live status is unavailable, continue with the deterministic fallback and the architecture explanation.
- Export under three minutes, confirm audible narration, and verify the public YouTube link in a private browser window.
