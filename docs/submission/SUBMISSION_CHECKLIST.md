# PolicyProof Submission Checklist

## Product and public access

- [ ] Builder completes the deterministic manual acceptance test
- [ ] One supervised GPT-5.6 fictional policy compilation is diagnosed successfully
- [ ] One supervised GPT-5.6 fictional document analysis is validated, if policy compilation succeeds
- [ ] Final English and French responsive review is complete
- [ ] Manual keyboard and screen-reader spot checks are complete
- [ ] Public application is deployed
- [ ] Public URL passes deterministic and status-endpoint smoke tests

## Repository

- [ ] Final local quality gates pass
- [ ] Final secret scan passes
- [ ] `.env.local`, build output, dependencies, and test artifacts remain ignored
- [ ] Repository has a selected license
- [ ] README setup and test instructions are current
- [ ] `REPOSITORY_REVIEW.md` pre-push and public-path checks are complete
- [ ] Deployment guide and post-deployment smoke tests are reviewed
- [ ] Fictional sample data and mocked evaluation contracts are included
- [ ] Codex contribution is explained
- [ ] GPT-5.6 responsibilities and deterministic responsibilities are explained
- [ ] Repository is committed and published to GitHub
- [ ] Public repository link works in a private browser window

## Video

- [ ] Video follows `DEMO_SCRIPT.md`
- [ ] Runtime is under three minutes
- [ ] Narration is audible and captions are reviewed if provided
- [ ] No credential, environment value, personal file, or private browser content is visible
- [ ] Deterministic fallback is demonstrated clearly
- [ ] Final screenshots contain no development indicator or real request reference
- [ ] Video is public on YouTube
- [ ] Public YouTube link works in a private browser window

## Submission form

- [ ] Project title and one-line pitch are final
- [ ] Work & Productivity category is selected
- [ ] English project description is final
- [ ] Architecture and meaningful GPT-5.6 use are explained accurately
- [ ] Limitations do not claim unverified live success
- [ ] Public application URL is included
- [ ] Public repository URL is included
- [ ] Public YouTube URL is included
- [ ] Primary Codex `/feedback` Session ID is included
- [ ] Final submission preview is reviewed before publishing
