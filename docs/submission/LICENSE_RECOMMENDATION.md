# License Recommendation

This is a practical project recommendation, not legal advice. The builder must make the final choice.

## MIT

MIT is short, familiar, permissive, and easy for hackathon judges and future contributors to understand. It permits reuse, modification, distribution, and commercial use while preserving the copyright and license notice. It contains no explicit patent grant.

## Apache License 2.0

Apache-2.0 is also permissive and adds an explicit patent license and patent-termination terms. It is longer and introduces notice obligations that deserve more careful repository administration.

## Recommendation

MIT is the simplest suitable default for this small solo prototype because PolicyProof has no separate patent policy or contributor program. Choose Apache-2.0 instead if explicit patent terms are important. Before publishing either license, the builder should confirm the preferred copyright name and consider legal advice if commercial or employer intellectual-property obligations may apply.
